# 隱私權政策 / Privacy Policy

## 中文

「譯」不會收集、儲存或傳送任何個人資料。

- 翻譯請求直接發送至 Google Translate（translate.googleapis.com），不經過任何第三方伺服器
- 使用者設定（目標語言、翻譯樣式）僅儲存在瀏覽器本機 storage，不會上傳
- 不追蹤使用者行為，不使用任何分析工具
- 不包含廣告

原始碼完全開源，可自行審查：https://github.com/vulcanshen/yi-translate

## English

"譯" does not collect, store, or transmit any personal data.

- Translation requests are sent directly to Google Translate (translate.googleapis.com) without passing through any third-party server
- User settings (target language, translation style) are stored only in the browser's local storage and are never uploaded
- No user behavior tracking, no analytics
- No ads

The source code is fully open source and available for review: https://github.com/vulcanshen/yi-translate
